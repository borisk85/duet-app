# Анализ конкурентов

Исследование проведено 07.04.2026.

## Глобальные игроки

### Vivino — лидер рынка, НЕ конкурент
- 70+ млн пользователей, 16 млн вин в базе
- Русский интерфейс есть, работает в СНГ
- **Почему не конкурент:** это маркетплейс для покупки вина. Food pairing — вторичная функция в каталоге, не главный сценарий. Пользователь сканирует этикетку, а не вводит блюдо.
- Монетизация: 15% комиссия с продаж + подписка $4-8/мес

### Hello Vino
- Только iOS, только английский
- Специализированный food pairing — Q&A интерфейс
- 6,685 отзывов в US App Store, в СНГ почти нет
- Только вино, без крепкого алкоголя

### Pocket Sommelier (OpenAI-powered)
- Фото блюда → 3 варианта вина через OpenAI Vision
- Новинка 2025, мало отзывов
- Только вино, только английский

### Sommo
- AI паиринг, фото блюда/этикетки
- Рейтинг 5.0, всего 2 отзыва — новинка
- Только английский

### Combivino
- Единственный с beer pairing (76 стилей пива)
- 900 типов вин + 2,000 рецептов
- Нет AI-генерации — статичная база
- Только английский

## СНГ-рынок

### ВИНТЕЛЛЕКТ (Vino.ru Telegram-бот) — ближайший, но НЕ конкурент
- Telegram-бот на базе YandexGPT, запущен июнь 2024
- Работает только внутри Telegram, только с каталогом одного магазина (Vino.ru)
- Только вино, только Россия
- **Почему важен:** доказывает спрос. Генерирует 10% трафика всего маркетплейса, каждая итерация дает +30% к продажам

### SimpleWine (сеть Simple)
- Российский ритейлер, 3,842 отзыва
- Каталог + покупка. Никакого AI-паиринга

### ВинЛаб
- Российский алкомаркет, 46,698 отзывов
- Только покупки и акции. Никакого food pairing

## Ключевой вывод

**Специализированного мобильного приложения для подбора алкоголя к еде на русском языке — не существует.**

Наше конкурентное преимущество:
1. Русский язык — первые на рынке СНГ
2. Все типы алкоголя (вино + крепкий + коктейли + пиво) — никто не делает
3. AI-генерация с объяснением "почему именно это" — не статичная база
4. История и избранное — подборки не теряются
5. Локализация по регионам (магазины, affiliate)
