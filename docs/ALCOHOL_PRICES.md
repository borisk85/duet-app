Используется как ориентир для бюджетной сегментации в AI-подборке.

## Примечание для AI-промптов

При сегменте "Бюджетно" — AI подбирает в нижнем диапазоне **для конкретного типа напитка**.
Не сравнивать пиво с виски по одной шкале — у каждого своя база.

Коктейли — отдельная история (стоимость ингредиентов), не включены в таблицу.
