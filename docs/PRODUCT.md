# Продуктовые решения — Дуэт

## Релевантность рекомендаций и локальные магазины

### Проблема

Claude может рекомендовать бренды которых нет в стране пользователя.
Например: картофельная водка Chopin есть в России, но отсутствует в Казахстане.
Рекомендация недоступного бренда = бесполезная рекомендация.

### Решение v1 — реализовано

#### 1. Фильтр доступности в промпте

В системном промпте Claude добавить инструкцию:

```
Рекомендуй только те бренды которые широко представлены
в регионе пользователя: {region}.

Казахстан: фокус на брендах доступных в Алматы и Астане.
Россия: фокус на брендах доступных в крупных городах РФ.
СНГ: универсальные бренды присутствующие во всех странах.

Избегай редких импортных брендов с ограниченной дистрибуцией
если они не представлены в регионе пользователя.
```

ALCOHOL_PRICES.md содержит правильные бренды по сегментам — использовать как ориентир.

#### 2. Кнопка "Найти в магазине" — локальный маркет по региону

| Регион | Маркет | Шаблон ссылки |
|--------|--------|---------------|
| Казахстан | Kaspi.kz | `https://kaspi.kz/shop/search/?q={brand_name}` |
| Россия | Winestyle.ru | `https://winestyle.ru/search/?search={brand_name}` |
| Россия (альтернатива) | Своевино | `https://swoevino.ru/search/?q={brand_name}` |
| СНГ (fallback) | Google Search | `https://www.google.com/search?q={brand_name}+купить+{city}` |

`brand_name` — URL-encoded. `city` — из профиля пользователя.

## Логика промпта (приоритеты)

При формировании рекомендации Claude учитывает:
1. **Регион** → доступность бренда
2. **Бюджет** → ценовой сегмент для данного типа напитка (см. ALCOHOL_PRICES.md)
3. **Предпочтения из профиля** → приоритет категорий в выдаче

Бюджетный сегмент — основной для большинства пользователей СНГ.
Средний — для особых случаев.
Премиум — редко, для экспериментов.

## Режим детализации ответа (из BUSINESS.md)

Передаётся в промпт как `detail_level`:

| Режим | Что получает пользователь |
|-------|--------------------------|
| simple | Тип напитка + краткое почему. "Красное сухое вино — подчеркнет вкус мяса." |
| standard | Бренд, сегмент, объяснение, совет по подаче. **Текущий по умолчанию.** |
| expert | Сорт винограда, регион, выдержка, температура, посуда, нюансы вкуса. |
