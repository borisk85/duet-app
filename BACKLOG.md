# Бэклог — идеи для v2+

Сюда идут хорошие идеи которые не берем в MVP. Не трогать до Google Play релиза.

## Главный экран

- **Умные чипы-подсказки** — вместо статичных примеров ("Говяжий стейк", "Суши") показывать популярные блюда из агрегированной истории всех пользователей (топ запросов по региону). Требует бэкенд + аналитику.
- **Бейджи на чипах** — показывать "🔥 Популярное" или "Сезонное" на чипах-подсказках для социального доказательства и геймификации.

## Результат

- **Умная приоритизация по кухне** — первым показывать напиток наиболее традиционный для кухни блюда (к японской — саке/пиво, к стейку — красное вино). Уже частично реализовано в промпте, проверить на разных кухнях.

## Монетизация

- **Баннерная реклама** — партнерство с локальными винными магазинами, только для Free-плана.
- **Affiliate v2** — кнопка "Найти в магазине" с affiliate-ссылкой по региону (Россия: Winestyle/Своевино, Казахстан: локальные сервисы).
- **Коллаборации с инфлюенсерами** — блогеры создают контент внутри приложения.

## Профиль / Онбординг

- **Онбординг** — 2-3 слайда при первом запуске.
- **Режим детализации** — simple/standard/expert переключатель (уже описан в BUSINESS.md).

## Продвижение

- **Product Hunt** — EN-аудитория, после международной версии.
- **EN-версия** — год 2, выход на глобальный рынок.
