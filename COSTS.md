# Затраты и расходы

## Разовые расходы (на старте)

| Статья | Стоимость | Когда |
|--------|-----------|-------|
| Apple Developer Account | $99/год | До публикации в App Store |
| Google Play Console | $25 разово | До публикации в Google Play |
| Figma (если нужен макет) | Бесплатно / $15/мес | На старте |

## Ежемесячные расходы

| Статья | Стоимость | Примечание |
|--------|-----------|------------|
| Railway (бэкенд + PostgreSQL) | $5-20/мес | Зависит от трафика |
| Claude Haiku API | ~$0.001/запрос | При 10,000 запросов/мес = $10 |
| Codemagic (iOS билд) | Бесплатно (500 мин/мес) → $45/мес | Бесплатного хватит на старте |
| RevenueCat | Бесплатно до $2,500 MRR | Потом 1% от дохода |
| Домен (опционально) | $15/год | Для лендинга или бэкенда |

## Итого первый год

| Период | Расходы |
|--------|---------|
| Старт (разовые) | ~$125 (Apple + Google) |
| Ежемесячно | $15-40/мес |
| Первый год | ~$300-600 |

**Сравнение с VELA:** примерно те же копеечные расходы. Основная инвестиция — время разработки.

## Инструменты разработки (бесплатные)

- Flutter SDK — бесплатно
- Dart — бесплатно (идет с Flutter)
- Android Studio — бесплатно
- VS Code + расширения — бесплатно
- Claude Code — уже есть

## Точка безубыточности

При расходах $30/мес нужно всего **15 платящих пользователей по $2/мес** чтобы выйти в ноль.
